## Version

### Native API

```c
const char *shadowhook_get_version(void);
```

Returns the ShadowHook version string (e.g., `"2.0.1-rc.10"`).

### Java API

```java
public static String ShadowHook.getVersion();
```

## Initialization

### Native API

```c
typedef enum {
  SHADOWHOOK_MODE_SHARED = 0, // Multiple hooks at same address, with proxy management and recursion prevention
  SHADOWHOOK_MODE_UNIQUE = 1, // Same address can only be hooked once
  SHADOWHOOK_MODE_MULTI  = 2  // Multiple hooks at same address, no automatic recursion prevention
} shadowhook_mode_t;

int shadowhook_init(shadowhook_mode_t default_mode, bool debuggable);
int shadowhook_get_init_errno(void);
```

`shadowhook_init()` returns `0` on success. A non-zero return value is an errno code. Only the first call takes effect; subsequent calls return the result of the first initialization.

`shadowhook_get_init_errno()` returns the errno from the most recent `shadowhook_init()` call.

### Java API

```java
package com.bytedance.shadowhook;

public class ShadowHook {
    public static int  init();
    public static int  init(Config config);
    public static int  getInitErrno();   // errno from the most recent init() call
    public static long getInitCostMs();  // wall-clock milliseconds taken by init()
}
```

Build a `Config` with `ConfigBuilder`:

```java
ShadowHook.init(new ShadowHook.ConfigBuilder()
    .setMode(ShadowHook.Mode.SHARED)
    .setDebuggable(false)
    .setRecordable(false)
    .setDisable(false)
    .build());
```

## Attributes

### Native API

```c
// Mode convenience macros
#define SHADOWHOOK_IS_SHARED_MODE (SHADOWHOOK_MODE_SHARED == shadowhook_get_mode())
#define SHADOWHOOK_IS_UNIQUE_MODE (SHADOWHOOK_MODE_UNIQUE == shadowhook_get_mode())
#define SHADOWHOOK_IS_MULTI_MODE  (SHADOWHOOK_MODE_MULTI  == shadowhook_get_mode())

shadowhook_mode_t shadowhook_get_mode(void);

bool shadowhook_get_debuggable(void);
void shadowhook_set_debuggable(bool debuggable);

bool shadowhook_get_recordable(void);
void shadowhook_set_recordable(bool recordable);

bool shadowhook_get_disable(void);
void shadowhook_set_disable(bool disable);
```

The default mode can only be set during initialization and cannot be changed afterward. Debuggable, recordable, and disable can be toggled at any time after initialization.

### Java API

```java
public static Mode    getMode();

public static boolean getDebuggable();
public static void    setDebuggable(boolean debuggable);

public static boolean getRecordable();
public static void    setRecordable(boolean recordable);

public static boolean getDisable();
public static void    setDisable(boolean disable);

public static String  getArch();  // returns "arm64-v8a", "armeabi-v7a", or empty string if not init'd
```

## Error handling

### Native API

```c
int         shadowhook_get_errno(void);
const char *shadowhook_to_errmsg(int error_number);
```

`shadowhook_get_errno()` returns a thread-local errno set by the most recent ShadowHook API call on the calling thread.

`shadowhook_to_errmsg()` returns a human-readable string for the given error number.

### Java API

```java
public static String toErrmsg(int errno);
```

## Hook API

### Hook flags

```c
#define SHADOWHOOK_HOOK_DEFAULT          0  // 0b0000 — use the default mode set at init
#define SHADOWHOOK_HOOK_WITH_SHARED_MODE 1  // 0b0001 — force shared mode for this hook
#define SHADOWHOOK_HOOK_WITH_UNIQUE_MODE 2  // 0b0010 — force unique mode for this hook
#define SHADOWHOOK_HOOK_WITH_MULTI_MODE  4  // 0b0100 — force multi mode for this hook
#define SHADOWHOOK_HOOK_RECORD           8  // 0b1000 — record lib/sym name (pass two extra char* args)
```

Multiple flags can be combined with `|`.

### Hooked callback typedef

```c
typedef void (*shadowhook_hooked_t)(
    int         error_number,
    const char *lib_name,
    const char *sym_name,
    void       *sym_addr,
    void       *new_addr,
    void       *orig_addr,
    void       *arg
);
```

Called after a pending hook (errno 1) completes when the target ELF is loaded into memory.

### Hook by function address

```c
// Default mode
void *shadowhook_hook_func_addr(void *func_addr, void *new_addr, void **orig_addr);

// Explicit mode + optional record names
void *shadowhook_hook_func_addr_2(
    void     *func_addr,
    void     *new_addr,
    void    **orig_addr,
    uint32_t  flags,
    ... /* char *record_lib_name, char *record_sym_name — required when SHADOWHOOK_HOOK_RECORD is set */
);
```

`func_addr` does not need to exist in the ELF symbol table.

### Hook by symbol address

```c
// Default mode
void *shadowhook_hook_sym_addr(void *sym_addr, void *new_addr, void **orig_addr);

// Explicit mode + optional record names
void *shadowhook_hook_sym_addr_2(
    void     *sym_addr,
    void     *new_addr,
    void    **orig_addr,
    uint32_t  flags,
    ... /* char *record_lib_name, char *record_sym_name */
);
```

`sym_addr` must exist in the ELF's `.dynsym` or `.symtab`.

### Hook by library name and symbol name

```c
// Default mode
void *shadowhook_hook_sym_name(
    const char  *lib_name,
    const char  *sym_name,
    void        *new_addr,
    void       **orig_addr
);

// Default mode + completion callback
void *shadowhook_hook_sym_name_callback(
    const char         *lib_name,
    const char         *sym_name,
    void               *new_addr,
    void              **orig_addr,
    shadowhook_hooked_t hooked,
    void               *hooked_arg
);

// Explicit mode
void *shadowhook_hook_sym_name_2(
    const char  *lib_name,
    const char  *sym_name,
    void        *new_addr,
    void       **orig_addr,
    uint32_t     flags
);

// Explicit mode + completion callback
void *shadowhook_hook_sym_name_callback_2(
    const char         *lib_name,
    const char         *sym_name,
    void               *new_addr,
    void              **orig_addr,
    uint32_t            flags,
    shadowhook_hooked_t hooked,
    void               *hooked_arg
);
```

If the target ELF is not yet loaded, these functions queue the task internally and return a non-NULL stub with errno 1 (pending). The hook executes automatically when the ELF is loaded, before linker calls `.init_array`.

**Return values:**
- Non-NULL, errno 0 — hook succeeded; save the stub for later unhook.
- Non-NULL, errno 1 — hook pending; ELF not yet loaded.
- NULL — hook failed; call `shadowhook_get_errno()` for details.

### Unhook

```c
int shadowhook_unhook(void *stub);
```

Returns `0` on success, `-1` on failure. Call `shadowhook_get_errno()` on failure.

## Intercept API

### Intercept flags

```c
#define SHADOWHOOK_INTERCEPT_DEFAULT                0  // 0b000 — general registers only
#define SHADOWHOOK_INTERCEPT_WITH_FPSIMD_READ_ONLY  1  // 0b001 — also read fpsimd registers
#define SHADOWHOOK_INTERCEPT_WITH_FPSIMD_WRITE_ONLY 2  // 0b010 — also write fpsimd registers
#define SHADOWHOOK_INTERCEPT_WITH_FPSIMD_READ_WRITE 3  // 0b011 — read and write fpsimd registers
#define SHADOWHOOK_INTERCEPT_RECORD                 4  // 0b100 — record lib/sym name
```

Reading and writing fpsimd registers is expensive. Use `DEFAULT` when fpsimd access is not needed.

### CPU context structs

```c
typedef union {
#if defined(__aarch64__)
  __uint128_t q;
#endif
  uint64_t d[2];
  uint32_t s[4];
  uint16_t h[8];
  uint8_t  b[16];
} shadowhook_vreg_t;

// arm64
#if defined(__aarch64__)
typedef struct {
  uint64_t          regs[31];   // x0–x30
  uint64_t          sp;
  uint64_t          pc;
  uint64_t          pstate;
  shadowhook_vreg_t vregs[32];  // q0–q31
  uint64_t          fpsr;
  uint64_t          fpcr;
} shadowhook_cpu_context_t;

// arm32
#elif defined(__arm__)
typedef struct {
  uint32_t          regs[16];   // r0–r15
  uint32_t          cpsr;
  uint32_t          fpscr;
  // NEON/VFPv4/VFPv3-D32: q0–q15 (d0–d31)
  // VFPv3-D16: q0–q7 (d0–d15)
  shadowhook_vreg_t vregs[16];
} shadowhook_cpu_context_t;
#endif
```

Inside the interceptor, all general registers can be read. All registers except `SP` and `PC` can be modified.

### Interceptor and intercepted-callback typedefs

```c
typedef void (*shadowhook_interceptor_t)(
    shadowhook_cpu_context_t *cpu_context,
    void                     *data
);

typedef void (*shadowhook_intercepted_t)(
    int                       error_number,
    const char               *lib_name,
    const char               *sym_name,
    void                     *sym_addr,
    shadowhook_interceptor_t  pre,
    void                     *data,
    void                     *arg
);
```

### Intercept by instruction address

```c
void *shadowhook_intercept_instr_addr(
    void                     *instr_addr,
    shadowhook_interceptor_t  pre,
    void                     *data,
    uint32_t                  flags,
    ... /* char *record_lib_name, char *record_sym_name */
);
```

`instr_addr` may be any instruction address inside a function, including mid-function addresses. For thumb instructions, ensure the address value is odd.

### Intercept by function address

```c
void *shadowhook_intercept_func_addr(
    void                     *func_addr,
    shadowhook_interceptor_t  pre,
    void                     *data,
    uint32_t                  flags,
    ... /* char *record_lib_name, char *record_sym_name */
);
```

### Intercept by symbol address

```c
void *shadowhook_intercept_sym_addr(
    void                     *sym_addr,
    shadowhook_interceptor_t  pre,
    void                     *data,
    uint32_t                  flags,
    ... /* char *record_lib_name, char *record_sym_name */
);
```

### Intercept by library name and symbol name

```c
void *shadowhook_intercept_sym_name(
    const char               *lib_name,
    const char               *sym_name,
    shadowhook_interceptor_t  pre,
    void                     *data,
    uint32_t                  flags
);

void *shadowhook_intercept_sym_name_callback(
    const char                *lib_name,
    const char                *sym_name,
    shadowhook_interceptor_t   pre,
    void                      *data,
    uint32_t                   flags,
    shadowhook_intercepted_t   intercepted,
    void                      *intercepted_arg
);
```

If the target ELF is not yet loaded, the intercept task is queued and a stub is returned with errno 1 (pending).

**Return values:**
- Non-NULL, errno 0 — intercept succeeded.
- Non-NULL, errno 1 — intercept pending; ELF not yet loaded.
- NULL — intercept failed.

### Unintercept

```c
int shadowhook_unintercept(void *stub);
```

Returns `0` on success, `-1` on failure.

## Symbol lookup

ShadowHook embeds [xDL](https://github.com/hexhacking/xDL) and exposes its symbol-lookup API, which bypasses linker namespace restrictions:

```c
void *shadowhook_dlopen(const char *lib_name);
void  shadowhook_dlclose(void *handle);

// Search .dynsym, then .symtab and .symtab-in-.gnu_debugdata
void *shadowhook_dlsym(void *handle, const char *sym_name);

// Search .dynsym only (fastest)
void *shadowhook_dlsym_dynsym(void *handle, const char *sym_name);

// Search .symtab and .symtab-in-.gnu_debugdata only
void *shadowhook_dlsym_symtab(void *handle, const char *sym_name);
```

Use `shadowhook_dlsym_dynsym()` when you know the symbol is in `.dynsym`. Use `shadowhook_dlsym_symtab()` for debug symbols. Use `shadowhook_dlsym()` for best compatibility across devices.

## Linker callbacks

<Note>
  Linker callbacks are only supported on Android 5.0 (API level 21) and above.
</Note>

```c
typedef void (*shadowhook_dl_info_t)(struct dl_phdr_info *info, size_t size, void *data);

// Register/unregister callbacks around .init + .init_array execution
int shadowhook_register_dl_init_callback(
    shadowhook_dl_info_t pre,
    shadowhook_dl_info_t post,
    void                *data
);
int shadowhook_unregister_dl_init_callback(
    shadowhook_dl_info_t pre,
    shadowhook_dl_info_t post,
    void                *data
);

// Register/unregister callbacks around .fini + .fini_array execution
int shadowhook_register_dl_fini_callback(
    shadowhook_dl_info_t pre,
    shadowhook_dl_info_t post,
    void                *data
);
int shadowhook_unregister_dl_fini_callback(
    shadowhook_dl_info_t pre,
    shadowhook_dl_info_t post,
    void                *data
);
```

`pre` and `post` cannot both be NULL. Returns `0` on success, `-1` on failure.

## Operation records

```c
// Item flags — combine with | or use SHADOWHOOK_RECORD_ITEM_ALL
#define SHADOWHOOK_RECORD_ITEM_ALL             0x7FF
#define SHADOWHOOK_RECORD_ITEM_TIMESTAMP       (1 << 0)
#define SHADOWHOOK_RECORD_ITEM_CALLER_LIB_NAME (1 << 1)
#define SHADOWHOOK_RECORD_ITEM_OP              (1 << 2)
#define SHADOWHOOK_RECORD_ITEM_LIB_NAME        (1 << 3)
#define SHADOWHOOK_RECORD_ITEM_SYM_NAME        (1 << 4)
#define SHADOWHOOK_RECORD_ITEM_SYM_ADDR        (1 << 5)
#define SHADOWHOOK_RECORD_ITEM_NEW_ADDR        (1 << 6)
#define SHADOWHOOK_RECORD_ITEM_BACKUP_LEN      (1 << 7)
#define SHADOWHOOK_RECORD_ITEM_ERRNO           (1 << 8)
#define SHADOWHOOK_RECORD_ITEM_STUB            (1 << 9)
#define SHADOWHOOK_RECORD_ITEM_FLAGS           (1 << 10)

// Returns malloc'd buffer — caller must free()
char *shadowhook_get_records(uint32_t item_flags);

// Writes records to fd — async-signal-safe, safe to call from signal handlers
void shadowhook_dump_records(int fd, uint32_t item_flags);
```

Records are ASCII text, one record per line separated by `\n`. Fields within each line are comma-separated in this order: TIMESTAMP, CALLER_LIB_NAME, OP, LIB_NAME, SYM_NAME, SYM_ADDR, NEW_ADDR, BACKUP_LEN, ERRNO, STUB, FLAGS.

## Macros

### `SHADOWHOOK_CALL_PREV`

Call the next function in the proxy chain (the original function, or the next proxy in the chain).

```c
// C++ source — first arg is current proxy function address, rest are call parameters
SHADOWHOOK_CALL_PREV(func, ...)

// C source — second arg is the function type (typedef), rest are call parameters
SHADOWHOOK_CALL_PREV(func, func_sig, ...)
```

Only valid in shared mode. Do not call the original function by name or via `orig_addr` in shared mode.

### `SHADOWHOOK_POP_STACK`

```c
// For C and C++
SHADOWHOOK_POP_STACK()
```

Must be called once before the proxy function returns in shared mode. Performs the internal hub stack cleanup. Must be called even if the original function is not called.

### `SHADOWHOOK_STACK_SCOPE`

```c
// For C++ only
SHADOWHOOK_STACK_SCOPE()
```

RAII alternative to `SHADOWHOOK_POP_STACK`. Place at the top of the proxy function; cleanup runs automatically at function exit.

### `SHADOWHOOK_ALLOW_REENTRANT`

```c
SHADOWHOOK_ALLOW_REENTRANT()
```

Allows the current proxy function to be reentered in shared mode. Use only when controlled recursion terminates under known business conditions.

### `SHADOWHOOK_DISALLOW_REENTRANT`

```c
SHADOWHOOK_DISALLOW_REENTRANT()
```

Restores the default reentrance-prevention behavior after `SHADOWHOOK_ALLOW_REENTRANT()`.

### `SHADOWHOOK_RETURN_ADDRESS`

```c
SHADOWHOOK_RETURN_ADDRESS()
```

Returns the original `LR` (return address) inside a shared-mode proxy function. Use this instead of `__builtin_return_address(0)`, because the shared-mode hub overwrites `LR` before calling the proxy.
