ShadowHook is a production-grade Android inline hook library from ByteDance. It lets you intercept and redirect function calls at runtime on armeabi-v7a and arm64-v8a devices running Android 4.1 through 16 (API 16–36).

## Key features

- **Stability** — battle-tested in ByteDance production apps at scale
- **Compatibility** — API and ABI backward-compatible across releases; supports Android 4.1–16
- **Three hook modes** — shared (chain + recursion-safe), multi (chain, no recursion guard), unique (exclusive)
- **Pending hooks** — auto-completes hooks on ELFs that haven't been loaded yet
- **Symbol resolution** — searches `.dynsym`, `.symtab`, and compressed `.gnu_debugdata` sections
- **Operation recording** — exportable audit log of every hook and intercept operation
- **CFI/FP unwind safe** — stack unwinding works correctly inside proxy and interceptor functions

<Note>
  ShadowHook is an **inline hook** library. If you need a PLT hook library for Android, see [ByteHook](https://github.com/bytedance/bhook).
</Note>
